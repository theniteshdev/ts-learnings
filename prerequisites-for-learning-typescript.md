## 1️⃣ Strong Fundamentals of JavaScript (Most Important)

You should be comfortable with:

- Variables and data types
- Functions and arrow functions
- Objects and arrays
- Scope and closures
- ES6+ features (let/const, spread, destructuring)
- Modules (import/export)
- Classes
- Basic asynchronous concepts (Promises, async/await)

If JavaScript feels confusing, TypeScript will feel even more confusing.

---

## 2️⃣ Basic Understanding of How JavaScript Runs

- How browsers execute JavaScript
- JavaScript is interpreted at runtime
- Basic idea of compilation vs interpretation

Because TypeScript introduces a compilation step, you need to understand what that means.

---

## 3️⃣ Comfort With Command Line and NPM (Basic Level)

- Basic Shell Commands
- How to install npm packages
- Using Aliases for Command Shortcuts (alias)

## 4️⃣ Basic concepts of Operating System

- What is a process
- How to inspect a process using System Informer App
- Environment Variables
